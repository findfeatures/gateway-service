# Gateway Service

The gateway service is the main API service for the backend.

Install the requirements into your virtualenv
```bash
pip install -e ".[dev]"
```

Run Unit Test
```bash
Make test
```

To run the service
```bash
Make run
```

(To install a pyenv virtualenv)
```bash
pyenv virtualenv -p python3.6 3.6.8 gateway-service
```
